## 📌 What is NeuralPath?

**NeuralPath** is a rigorously structured, fully free AI/ML curriculum — designed to take a complete beginner from *zero Python knowledge* all the way to **publishing their first ML research paper**. No subscriptions. No paywalls. No GPU bills. Every single tool, dataset, and compute resource is either open-source or permanently free.

This is not a bootcamp. It is not a MOOC. It is a **living, self-paced learning path** built with a research-first mindset — the same way professional ML engineers and researchers actually learn.

<br />

## 🎯 Curriculum Overview

> **10 Phases · 48 Weeks · $0 Cost**
```
PHASE 01 ── Python, Colab & Dev Environment       Weeks  1– 3
PHASE 02 ── NumPy, Pandas & Visualization          Weeks  4– 6
PHASE 03 ── Data Cleaning, EDA & Feature Eng.     Weeks  7– 9
PHASE 04 ── Classical Machine Learning             Weeks 10–15
PHASE 05 ── Deep Learning with PyTorch             Weeks 16–22
PHASE 06 ── NLP & Transformers                     Weeks 23–28
PHASE 07 ── Large Language Models (LLMs)           Weeks 29–34
PHASE 08 ── MLOps & Free Deployment                Weeks 35–38
PHASE 09 ── Reading & Conducting AI Research       Weeks 39–42
PHASE 10 ── Thesis Writing & First Publication     Weeks 43–48
```

<br />

**Phase 01 — Python, Colab & Dev Environment** *(Weeks 1–3)*
- Python fundamentals: variables, loops, OOP, file I/O, generators
- Google Colab GPU runtime setup, Kaggle free GPU & datasets
- Git version control, GitHub, ML project structure & documentation

**Phase 02 — NumPy, Pandas & Visualization** *(Weeks 4–6)*
- NumPy: arrays, broadcasting, linear algebra (matmul, einsum), vectorization
- Pandas: DataFrame, GroupBy, merge/joins, pivot, missing value handling
- Matplotlib & Seaborn: publication-quality charts, KDE, violin plots

**Phase 03 — Data Cleaning, EDA & Feature Engineering** *(Weeks 7–9)*
- MCAR/MAR/MNAR missing data patterns, outlier detection & treatment
- Encoding (ordinal, one-hot, target), scaling, datetime features
- Automated EDA with ydata-profiling & Sweetviz

**Phase 04 — Classical Machine Learning** *(Weeks 10–15)*
- Regression: Linear, Ridge, Lasso, ElasticNet; bias-variance tradeoff
- Classification: Logistic Regression, KNN, SVM, Decision Trees, Naive Bayes
- Ensemble: Random Forest, XGBoost, LightGBM, CatBoost, Stacking & Blending
- Unsupervised: K-Means, DBSCAN, PCA, t-SNE, UMAP
- Hyperparameter tuning with Optuna

**Phase 05 — Deep Learning with PyTorch** *(Weeks 16–22)*
- Neural networks from scratch: backprop, activations, autograd
- CNNs: ResNet, transfer learning, data augmentation (Colab GPU)
- RNN/LSTM/GRU, Seq2Seq, attention basics
- Training best practices: batch norm, dropout, LR schedulers, mixed precision

**Phase 06 — NLP & Transformers** *(Weeks 23–28)*
- Tokenization, TF-IDF, Word2Vec, GloVe embeddings
- Transformer deep dive: self-attention, positional encoding, BERT & GPT
- HuggingFace: fine-tune BERT, Datasets API, push models to Hub (free)

**Phase 07 — Large Language Models** *(Weeks 29–34)*
- GPT architecture, prompt engineering, run Llama/Mistral locally with Ollama
- Fine-tuning: LoRA, QLoRA, prefix tuning, RLHF basics (free Colab GPU)
- RAG systems: Chroma/FAISS vector DBs, LangChain, LlamaIndex

**Phase 08 — MLOps & Free Deployment** *(Weeks 35–38)*
- Experiment tracking: MLflow (open-source), Weights & Biases (free tier)
- DVC for data versioning, automated ML pipelines
- Deployment: FastAPI, Docker, HuggingFace Spaces, Streamlit (all free)

**Phase 09 — Reading & Conducting AI Research** *(Weeks 39–42)*
- How to read ML papers fast, arXiv navigation & email digest alerts
- Hypothesis design, ablation studies, statistical significance testing
- Literature reviews with Connected Papers & Semantic Scholar

**Phase 10 — Thesis Writing & First Publication** *(Weeks 43–48)*
- LaTeX on Overleaf (free): IEEE & ACL templates, BibTeX, figures, equations
- Full capstone research project with experiments & ablation study
- Submit preprint to arXiv, OpenReview workshop submissions

</details>

## 🚀 Getting Started

### Run Locally
```bash
git clone https://github.com/nasif120696-collab/neuralpath.git
cd neuralpath
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Deploy on GitHub Pages (Free)
```
1. Repository → Settings → Pages
2. Source: Deploy from branch → main → / (root)
3. Save → live at: https://nasif120696-collab.github.io/neuralpath
```

### Deploy on Netlify (Free)
```
1. netlify.com → New site from Git → connect repo
2. Build command: (empty)   Publish directory: /
3. Deploy — live in ~30 seconds
```
